About

I'm a Web & Software Developer working for Deltatre, from their Turin offices. I work both on the back-end and front-end services and I consider myself a full-stack dev. I graduated from Integrated Backend Services course, hosted by ITS ICT - Turin, where I started to dig into development.

  • My stack
  • C# and .NET Core
  • Typescript & React
  • Docker
  • MongoDb
  • Git
  • Azure DevOps & Github

Would I like to work with other langs? Sure I would! I'd be happy to study and work with Rust & Go while I wouldn't be keen to work with PHP and Java.

Why? I'm not a jack-of-all-trades fan. I love to learn and master, better than knowing something of everything. Rust or Go would be domains that help me learn and challenge myself on new concepts, different perspectives, low-level topics I didn't study previously. I love to learn, as already said. PHP or Java, on the other side... are C#-like. I like C# more and I prefer to master it, instead of using them all without choosing my bestie. If someone needs PHP, I think it's best to actually hire a PHP dev than making me mock one.

I love programming and I want to grow and learn as much as I can as a developer. I'm trying to learn more about Design Patterns and Software Architecture, to improve my capabilities of creating good, clean, better code. For the same reason, I want to learn and improve over TDD, as I use it but I'd like to become better at it.

Oh, don't forget - the tech thing I hate the most is CSS and everything concerning. If you want me to mostly work on Css-things - well, you're messing with the wrong guy mate. ðŸ˜¤
